# Jogo do Número Secreto

Projeto desenvolvido durante o curso de Lógica de Programação da Alura com o objetivo de praticar os conceitos fundamentais de JavaScript.

## Sobre o projeto

O jogo gera um número aleatório entre **1 e 1000**, e o jogador deve tentar adivinhar qual é o número secreto. A cada tentativa, o programa informa se o número secreto é maior ou menor que o valor digitado até que o jogador acerte.

Ao final, o jogo exibe a quantidade de tentativas utilizadas.

## Conceitos praticados

* Variáveis e constantes;
* Estruturas condicionais (`if` e `else`);
* Laço de repetição (`while`);
* Operadores de comparação;
* Geração de números aleatórios com `Math.random()`;
* Conversão de valores com `parseInt()`;
* Operador ternário;
* Manipulação de entrada e saída com `prompt()` e `alert()`.

## Tecnologias utilizadas

* JavaScript

> **Observação:** O HTML e o CSS foram disponibilizados pelo curso e não sofreram alterações. O foco deste projeto foi a implementação da lógica em JavaScript.

## Objetivo

Praticar os fundamentos da programação e desenvolver o raciocínio lógico por meio da criação de um jogo simples de adivinhação.
